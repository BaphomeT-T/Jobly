# Jobly Landing Page

A modern, responsive landing page built to match the Figma design exactly.

## Features

- **Header**: Logo, navigation, and login button
- **Subheader**: Navigation items with dividers
- **Banner**: Promotional banner with "Exclusive offers" heading
- **Job Cards**: Interactive job listing cards with company information, job details, and tags
- **Hero Section**: Call-to-action section with overlay background
- **Footer**: Language selector and footer content

## Design Specifications

### Colors
- Primary Shade 10: `#3B1329`
- Primary Shade 60: `#892753`
- Primary Tint 10: `#FFF0F5`
- Neutral shades: Various grays for borders and text
- White: `#FFFFFF`
- Typography: `#141514`

### Typography
- **Headings**: Poppins, 600 weight, 32px
- **Subtitles**: Poppins, 500 weight, 20px
- **Body**: Inter, 500 weight, 14px
- **Buttons**: Poppins, 600 weight, 16px

### Layout
- Container width: 1440px
- Header height: 86px
- Card border radius: 4px
- Tag border radius: 24px
- Spacing: 4px, 8px, 16px, 24px, 32px, 56px
